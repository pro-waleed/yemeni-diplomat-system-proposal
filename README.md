# النظام الدبلوماسي المتكامل

نموذج تنفيذي تجريبي عربي لنظام متكامل لإدارة العمل الدبلوماسي في وزارة الخارجية اليمنية، ومهيأ للنشر مباشرة عبر `GitHub Pages`.

## محتوى المشروع

- `index.html`: واجهة النموذج التنفيذي التجريبي.
- `styles.css`: التنسيق والهوية البصرية للتطبيق.
- `app.js`: البيانات التجريبية ومنطق التفاعل بين الوحدات.
- `.github/workflows/deploy-pages.yml`: نشر تلقائي إلى GitHub Pages عند الدفع إلى الفرع الرئيسي.

## فكرة النموذج

يحّول هذا المشروع المقترح المكتبي إلى نسخة قابلة للعرض والتجريب أمام قيادة الوزارة، ويغطي:

- لوحة قيادة تنفيذية.
- وحدات التقارير والتعاميم والاجتماعات والخطط.
- ملف البعثات والدوائر.
- منصة تدريب تجريبية.
- شاشة حوكمة وصلاحيات.
- بيانات نموذجية وتفاعلات أساسية للتنقل والعرض.

## التشغيل المحلي

يمكن فتح الملف [index.html](/E:/MyProject/Yemeni_Diplomat/index.html) مباشرة في المتصفح، أو استخدام خادم محلي بسيط:

```powershell
cd E:\MyProject\Yemeni_Diplomat
npx serve .
```

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع ملفات هذا المجلد إلى المستودع.
3. تأكد أن الفرع الافتراضي هو `main`.
4. سيعمل ملف GitHub Actions تلقائيًا على نشر الموقع.
5. بعد اكتمال النشر سيظهر الرابط بالشكل:

```text
https://<github-username>.github.io/<repository-name>/
```

## ملاحظات تطوير لاحقة

- يمكن ربط البيانات لاحقًا بواجهة خلفية وقاعدة بيانات.
- يمكن إضافة تسجيل دخول وصلاحيات حقيقية.
- يمكن تحويل كل وحدة إلى شاشة تشغيل كاملة مع نماذج إدخال واعتماد.
