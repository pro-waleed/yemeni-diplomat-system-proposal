# النظام الدبلوماسي المتكامل

عرض عربي تنفيذي لمقترح نظام متكامل لإدارة العمل الدبلوماسي في وزارة الخارجية اليمنية، ومهيأ للنشر مباشرة عبر `GitHub Pages`.

## محتوى المشروع

- `index.html`: الصفحة الرئيسية للعرض التنفيذي.
- `styles.css`: التنسيق والهوية البصرية للموقع.
- `.github/workflows/deploy-pages.yml`: نشر تلقائي إلى GitHub Pages عند الدفع إلى الفرع الرئيسي.

## فكرة الموقع

يحّول هذا المشروع المقترح المكتبي إلى عرض مؤسسي قابل للمشاركة مع قيادة الوزارة، ويغطي:

- الرؤية والأهداف.
- الوحدات التشغيلية الأساسية للنظام.
- الحوكمة والصلاحيات والأمن.
- خارطة التنفيذ المرحلية.
- مؤشرات الأداء والملحقات التنفيذية.

## التشغيل المحلي

يمكن فتح الملف [index.html](/E:/MyProject/Yemeni_Diplomat/index.html) مباشرة في المتصفح، أو استخدام خادم محلي بسيط:

```powershell
cd E:\MyProject\Yemeni_Diplomat
npx serve .
```

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع ملفات هذا المجلد إلى المستودع.
3. تأكد أن الفرع الافتراضي هو `main`.
4. سيعمل ملف GitHub Actions تلقائيًا على نشر الموقع.
5. بعد اكتمال النشر سيظهر الرابط بالشكل:

```text
https://<github-username>.github.io/<repository-name>/
```
